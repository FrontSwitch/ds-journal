# Somatic Mapping & System State Trackers

Core Logic: Use timestamped, tactile input (large buttons) to record real-time signals without requiring voice logs or post-event recall. 

## Nudge

Use a haptic nudge (vibration) every 5–15 minutes to trigger a check-in.

## Module 1: Run-Phase Signal Tracker

+ Steady: Rhythmic breathing; system in sync.
+ HR Drift: Target exceeded; forgot to track.
+ Sidetracked: Environmental distraction (e.g., mushrooms).
+ Heavy Legs: High mechanical fatigue/load.
+ Laboring: High oxygen demand; heavy breathing.
+ Monster Mode: Controlled release of power; the "Kraken."

## Module 2: Therapy-Phase Signal Tracker

+ Grounded: Present and connected to the dialogue.
+ Foggy: Dissociative drift; losing the thread.
+ High Latency: Access control is blocking a topic.
+ Static: Overwhelming noise or internal interference.
+ Spike: Sharp physical or internal hit.
+ Offline: Full shutdown or Gatekeeper intervention.

## Output Visualization: X-axis (Time) vs. Y-axis (Mode).

## Where else would this be useful?

Emotion/System State Panel
+ Neutral/Clear: System is operational; no significant interference.
+ Buzzing: High-frequency anxiety or physical restlessness.
+ Low Battery: Physical or emotional exhaustion; system scaling back.
+ Guarded: Access control is high; internal walls are active.
+ Turbulent: High volatility or rapid state shifting.
+ Muted: Emotional numbness or detachment; "flat-lining."
